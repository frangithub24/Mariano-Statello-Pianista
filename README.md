# Mariano Statello - Professional Musician Website

A responsive website for professional pianist and composer Mariano Statello, featuring his music, sheet music, tutorials, and videos.

🔗 **Live Site:** https://frangithub24.github.io/Mariano-Statello-Pianista/

## Features

- **Responsive Design** - Works on mobile, tablet, and desktop
- **Audio Player** - Continuous playback across pages using localStorage
- **Mobile Navigation** - Hamburger menu with blur effect
- **Contact Form** - Integrated with Formspree
- **Multiple Sections:**
  - Biography
  - Sheet Music (Partituras)
  - Video Tutorials
  - Video Gallery
  - Contact Form

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM manipulation, localStorage)
- GitHub Pages

## Key Features Implementation

- **Persistent Audio Playback** - Music continues playing when navigating between pages
- **Responsive Navigation** - Mobile-friendly hamburger menu with smooth animations
- **Mobile-First Design** - Optimized for all screen sizes

## Client

Mariano Statello - Musician, Pianist, and Composer from Argentina
